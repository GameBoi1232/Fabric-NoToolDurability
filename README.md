# NoToolDurability

A simple mod for Minecraft which disables durability. Mod can be toggeled in game using the /notooldurability command.

## Commands

/notooldurability enable >>> Tools **DO NOT** take durability damage

/notooldurability disable >>> Tools **DO** take durability damage

/notooldurability status >>> shows if tools do or not take durability damage
